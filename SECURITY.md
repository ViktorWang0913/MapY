# Security Policy

## Reporting

Please report security issues by email:

```text
mapy_zstudio@163.com
```

Do not publish exploit details in public issues before the problem is reviewed.

## Sensitive Files

Do not commit:

- `.env` files
- SMTP authorization codes
- private API tokens
- local user files
- generated installers
- build output

The repository includes `.env.example` only as a placeholder reference.
