## 下载

Windows 和 macOS 安装包请从 [GitHub Releases](https://github.com/ViktorWang0913/MapY/releases/latest) 下载。

## 开发

```bash
npm install
npm run dev
```

桌面开发：`npm run tauri:dev`  
打包安装包：`npm run tauri:build`
